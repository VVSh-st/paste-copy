# Slash palette fix

Затронутые файлы:
- blocks.js
- styles.css

Изменения:
- Slash-палитра `/` теперь показывает быстрые команды первыми, чтобы дефолтные сниппеты не вытесняли commands из лимита.
- Добавлены секции "Быстрые команды" и "Сниппеты".
- Переписана разметка строк палитры: иконка, заголовок, короткий preview значения.
- Добавлены стили `.slash-palette`, `.slash-item`, `.slash-section-label` в стиле текущего UI.
- Добавлен guard на отложенное позиционирование после закрытия палитры.

Проверки:
- Статически проверены селекторы: `.slash-palette`, `.dropdown-item`, `.slash-item`, `.slash-section-label`.
- Логика больше не ограничивает пустой `/` первыми 12 сниппетами; команды попадают в меню сразу.
