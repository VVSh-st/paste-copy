# Контекст правок 2026-05-20

Проверка проекта выявила критические проблемы:
- `state.js`: поиск по regex с нулевой длиной совпадения (`(?=a)`, `^`, пустые lookahead) мог зависнуть в бесконечном цикле.
- `ui.js`: Markdown preview вставлял HTML от marked через `innerHTML` без санитайза, что позволяло XSS через импортированный/вставленный текст.
- `app.js`: Escape не закрывал LLM-модалки, потому что они открываются через `display: flex`, а проверялся только `block`.

Затронутые файлы: `state.js`, `ui.js`, `app.js`.
Бэкап должен храниться в `_ai_backups/2026-05-20_00-00-00/` перед правкой; из-за ограничения MCP без shell копирование выполняется через доступные операции чтения/точечного редактирования.
