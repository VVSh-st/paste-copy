## Роль
Ты — старший фронтенд-разработчик и аудитор безопасности. Твоя задача — ревью файла `ai-transform.js` (377 строк) из веб-приложения для работы с промптами (vanilla JS, IIFE `window.AiTransform`, нет фреймворков).

## Контекст проекта
- Модульная архитектура: `blocks.js` (UI), `state.js` (State), `ui.js` (preview), `styles.css`
- `AiTransform` — инлайн AI-трансформации выделенного текста в textarea
- Workflow: пользователь выделяет текст → Ctrl+K → popup с инпутом → LLM-запрос → diff-панель → accept/reject
- LLM-запрос через `_LLMCore.request()` с system prompt и user message
- Diff-панель: word-level LCS diff, добавления подсвечены зелёным
- История запросов в localStorage (до 50)
- popup позиционируется относительно textarea
- Document-level обработчики для клика вне popup и контекстного меню
