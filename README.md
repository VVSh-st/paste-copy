# paste\copy

<!-- GIF / screenshot сюда -->

Браузерный редактор промптов для работы с LLM. Собирай сложные промпты из блоков, копируй в один клик, не теряй версии.

## Зачем

Потому что копипастить промпт из заметок в чат — это боль. А когда промпт из 10 блоков и ты меняешь один — боль двойная.

paste\copy позволяет:
- Разбить промпт на логические блоки (роль, контекст, задача, ограничения)
- Включать/выключать отдельные блоки из итогового текста
- Копировать собранный промпт одной кнопкой
- Хранить несколько промптов на вкладках

## Быстрый старт

1. Открой `index.html` в браузере
2. Создай вкладку (`+`)
3. Добавь блоки (`+ Добавить блок`)
4. Напиши промпт
5. Нажми «Скопировать превью» → вставь в чат с LLM

Сервер не нужен. Установка не нужна. Python не нужен.

## Возможности

### Блоки

| Тип | Описание |
|-----|----------|
| 📝 Текст | Основной блок с Markdown-превью |
| 📌 Sticky | Цветная заметка (не в превью) |
| ☑️ TODO | Чеклист с подзадачами |
| 📊 Таблица | Структурированные данные |
| 🧩 Переменные | `{{name}}` — меняешь一处, обновляется везде |
| 🧱 Сниппеты | Быстрые команды и шаблоны |

### Инструменты

- **Переводчик** — перевод прямо в блоке (Google, Microsoft, Tencent)
- **AI-трансформация** — выдели текст → попроси LLM изменить
- **Текстовый линтер** — убирает мусор без нейросетей
- **Автодополнение** — подсказки при наборе (работает из Gist)
- **Визуальный Diff** — сравнивай версии с подсветкой
- **Якоря** — закладки на нужные места
- **Prompt Loom** — сниппеты и переменные через `\`

### Синхронизация

- **GitHub Gist** — Push/Pull между устройствами
- **Снапшоты** — история версий с возможностью отката
- **Локальный бэкап** — автоматическое сохранение

### LLM-интеграция

- Мини-чат — спроси модель не покидая редактор
- Оценка промпта — 5 критериев с анимированной шкалой
- Сжатие токенов — убери лишнее без потери смысла
- Автозаголовок — LLM придумает название для блока

## Горячие клавиши

| Комбинация | Действие |
|------------|----------|
| `Ctrl + S` | Сохранить |
| `Ctrl + Z` / `Y` | Отмена / Повтор |
| `Ctrl + F` | Поиск и замена |
| `Ctrl + T` | Новая вкладка |
| `Ctrl + W` | Закрыть вкладку |
| `Ctrl + D` | Дублировать вкладку |
| `Ctrl + Shift + C` | Копировать превью |
| `F2` | Справка |
| `Alt + L` | Мини-чат |

## Технологии

- Vanilla JS (без фреймворков)
- CSS (без препроцессоров)
- HTML
- marked.js + highlight.js (CDN)

Всё работает в браузере. Данные хранятся в localStorage + GitHub Gist.
