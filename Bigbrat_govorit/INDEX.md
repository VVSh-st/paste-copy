## Итог по Task 2

Ключевые рекомендации всех аудиторов:
1. **Адаптивный порог** — не фиксированные 8000, а % от контекстного окна модели
2. **Web Worker** — парсинг 100K+ блокирует UI
3. **Лемматизация** — для русского языка (дубли: "тестирование", "тест", "тесты")
4. **Compression levels** — light/medium/aggressive
5. **Кэширование** — hash(text) → skeleton
6. **Защита от отрицаний** — "не React" не должно стать ключевым термином "react"
