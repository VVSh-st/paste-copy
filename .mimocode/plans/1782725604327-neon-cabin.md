# Аудит-баги: Unicode word boundary + Дедупка сниппетов

## Контекст

Биг брат провёл аудит проекта. Два подтверждённых бага:
1. `\b` в regex не понимает кириллицу — поиск "целое слово" не работает для русского текста
2. Дедупка сниппетов в пикере не нормализует локальные блоки → видимые дубликаты

Архитектурный риск (setLayout shallow merge) — **пропускаем**, нет активного бага.

---

## Баг #1: Unicode word boundary

### Проблема

`\b` в JS regex считает word-символом только `[A-Za-z0-9_]`. Кириллица не входит в этот класс → для русского текста `\b` всегда возвращает `null`.

### 3 места для исправления

| # | Файл | Строка | Функция | Текущий код |
|---|------|--------|---------|-------------|
| 1 | `state.js` | 755 | `makeSearchRe` | `` `\b${pattern}\b` `` |
| 2 | `ui.js` | 1124 | `makeRe` | `` `\b${pattern}\b` `` |
| 3 | `ui.js` | 1240 | `renderResults` | `` `\b${escaped}\b` `` |

### Фикс

Заменить `\b${X}\b` на Unicode-aware lookaround:

```js
// Было:
if (options.wholeWord) pattern = `\\b${pattern}\\b`;
return new RegExp(pattern, flags);

// Стало:
if (options.wholeWord) pattern = `(?<![\\p{L}\\p{N}_])${pattern}(?![\\p{L}\\p{N}_])`;
return new RegExp(pattern, flags + 'u');
```

Флаг `u` обязателен для `\p{L}` и `\p{N}`. Lookaround нулевой ширины — не ломает `lastIndex`.

## Баг #2: Дедупка сниппетов

### Проблема

В `getAllSnippetsAndCommands()` (state.js ~878-898):
- Глобальные сниппеты: `seen.has(normalizeSnippetValue(item.value))` ✓
- Локальные сниппеты/команды блоков: `seen.has(i.value)` ✗ — без нормализации

Результат: один и тот же текст с разными пробелами/`\r\n` → дубликаты в пикере `!`.

### Фикс

Добавить `normalizeSnippetValue()` перед `seen.has`/`seen.add` для локальных блоков.

```js
// Было (строки ~887-898):
for (const [bid, b] of blocks.entries()) {
  if (b.snippets) b.snippets.forEach(i => {
    if (!seen.has(i.value)) { seen.add(i.value); result.push({...}); }
  });
  if (b.commands) b.commands.forEach(i => {
    if (!seen.has(i.value)) { seen.add(i.value); result.push({...}); }
  });
}

// Стало:
for (const [bid, b] of blocks.entries()) {
  if (b.snippets) b.snippets.forEach(i => {
    const nv = normalizeSnippetValue(i.value);
    if (!seen.has(nv)) { seen.add(nv); result.push({...}); }
  });
  if (b.commands) b.commands.forEach(i => {
    const nv = normalizeSnippetValue(i.value);
    if (!seen.has(nv)) { seen.add(nv); result.push({...}); }
  });
}
```
