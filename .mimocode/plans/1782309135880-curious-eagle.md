# Inline Diff для именованных версий (снапшотов)

## Суть
Добавить кнопку "Сравнить" к каждой сохранённой версии. При нажатии — показать inline diff выбранной версии vs текущее состояние, с подсветкой удалений (красным) и вставок (зелёным).

## Что уже есть
- LCS-based diff алгоритм в `llm-features.js:497-614` (computeByToken, computeByLine, trimCommonEdgesFallback, mergeAdjacentOps) — возвращает `[{ type: 'eq'|'ins'|'del', text }]`
- CSS классы `.diff-del` / `.diff-ins` в `styles.css:4136-4158` — подсветка удалений/вставок
- Модуль Snapshots в `ui.js:872-999` — рендер списка снапшотов
- HTML модалки в `index.html:855-883`
- State API: `serialize()`, `getNamedSnapshots()`, `restoreNamedSnapshot()`, `deleteNamedSnapshot()`
- Структура данных: снапшот = `JSON.stringify({ blocks, separator })`, блок = `{ type: 'text', subtabs: [{ label, value }], title }`

## План

### 1. Новый модуль `diff-engine.js` (~120 строк)
Вынести diff-логику в отдельный модуль, переиспользуя алгоритм из llm-features.js:

```js
const DiffEngine = (() => {
  // computeByToken, computeByLine, trimCommonEdgesFallback, mergeAdjacentOps — из llm-features.js
  // + extractTextFromSnapshot(snapshotData) — извлекает склеенный текст всех text-блоков
  // + renderInlineDiff(ops) — возвращает HTML строку с <span class="diff-del"> и <span class="diff-ins">
  return { computeDiff, renderInlineDiff, extractTextFromSnapshot };
})();
```

Ключевые функции:
- `extractTextFromSnapshot(data)` — парсит `data` (JSON), проходит по blocks, собирает `b.title + ': ' + subtabs[activeSubtab].value` для каждого text-блока с непустым содержимым. Возвращает строку.
- `computeDiff(oldText, newText)` — LCS diff, возвращает ops
- `renderInlineDiff(ops)` — escapes HTML, оборачивает del/ins в `<span class="diff-del">` / `<span class="diff-ins">`

### 2. Модификация `ui.js` — Snapshots модуль
- Добавить кнопку "🔍 Сравнить" в `renderList()` рядом с "↩ Восстановить"
- Добавить панель diff (`<div id="snap-diff-panel">`) под списком снапшотов
- При клике "Сравнить": 
  1. Получить `State.serialize()` → извлечь текущий текст
  2. Парсить снапшот → извлечь текст версии
  3. `DiffEngine.computeDiff(oldText, newText)`
  4. `DiffEngine.renderInlineDiff(ops)` → вставить в панель
- Кнопка "Закрыть diff" скрывает панель

### 3. Модификация `index.html`
- Добавить `<div id="snap-diff-panel" class="snap-diff-panel" style="display:none"></div>` внутрь модалки snapshots, после `#snapshots-list`

### 4. Модификация `styles.css`
Добавить стили для панели diff:
- `.snap-diff-panel` — скроллируемый блок с моноширинным шрифтом, border, padding
- `.snap-diff-header` — шапка с заголовком и кнопкой закрытия
- Переиспользовать существующие `.diff-del` / `.diff-ins`

### 5. Подключение `diff-engine.js` в `index.html`
Добавить `<script src="diff-engine.js"></script>` перед `ui.js` (порядок: state → diff-engine → ui).

## Верификация
1. Открыть `index.html` в браузере
2. Ввести текст в блок, сохранить снапшот "Версия 1"
3. Изменить текст, сохранить снапшот "Версия 2"
4. Нажать "🔍 Сравнить" на "Версия 1" — должен показать diff текущего vs Версия 1
5. Проверить: удалённый текст подсвечен красным с зачёркиванием, добавленный — зелёным
6. Кнопка "Закрыть" скрывает панель diff
7. Проверить с пустыми блоками и блоками без текста
