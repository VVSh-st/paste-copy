# Flowchart — визуальный редактор блок-схем

## Концепция

Новый модуль `flowchart.js` — визуальный редактор блок-схем, по аналогии с MindMap. LLM генерирует JSON с нодами и рёбрами, рендерит SVG, пользователь может перетаскивать ноды. Всё лучшее из MindMap: glass-panel, zoom/pan, parallax, stagger animation, bloom, proximity controls.

## Этап 2: Промпт LLM

### Формат JSON

```json
{
  "nodes": [
    { "id": "n1", "label": "Начало", "shape": "stadium" },
    { "id": "n2", "label": "Ввод данных", "shape": "rect" },
    { "id": "n3", "label": "Условие?", "shape": "diamond" },
    { "id": "n4", "label": "Да", "shape": "rect" },
    { "id": "n5", "label": "Нет", "shape": "rect" },
    { "id": "n6", "label": "Конец", "shape": "stadium" }
  ],
  "edges": [
    { "from": "n1", "to": "n2", "label": "" },
    { "from": "n2", "to": "n3", "label": "" },
    { "from": "n3", "to": "n4", "label": "Да" },
    { "from": "n3", "to": "n5", "label": "Нет" },
    { "from": "n4", "to": "n6", "label": "" },
    { "from": "n5", "to": "n6", "label": "" }
  ]
}
```

### Формы нод (shape)

| shape | Отрисовка | Описание |
|-------|-----------|----------|
| `rect` | Прямоугольник rx=8 | Процесс, действие |
| `round` | Скруглённый rect rx=20 | Начало/конец (альтернатива) |
| `stadium` | Скруглённый rect rx=40 | Начало/конец |
| `diamond` | Ромб | Условие, решение |
| `circle` | Круг | Соединитель |
| `cylinder` | Прямоугольник с дугой | База данных |

### Промпт в llm-core.js

```
flowchart: 'You are a flowchart designer. ## Этап 3: Рендер нод

### Базовые размеры

```js
const NODE_DEFAULTS = { w: 160, h: 50 };
function _nodeSize(node) {
  const lines = (node.label || node.id).split('\n');
  const maxLineW = Math.max(...lines.map(l => l.length)) * 7 + 32;
  const w = Math.max(NODE_DEFAULTS.w, maxLineW);
  const h = NODE_DEFAULTS.h + Math.max(0, (lines.length - 1) * 16);
  switch (node.shape) {
    case 'diamond':  return { w: Math.max(w, 100), h: Math.max(h, 70) };
    case 'circle':   return { s: Math.max(40, w * 0.6) }; // s = diameter
    case 'cylinder': return { w, h: h + 14 };
    default:         return { w, h };
  }
}
```

### Отрисовка по shape

- **rect**: `<rect rx="8">` + `fill="color+12"` + `stroke="color+40"`
- **stadium**: `<rect rx="40">` + тот же fill/stroke
- **diamond**: `<polygon>` из 4 точек (cx±w/2, cy±h/2)
- **circle**: `<circle r="s/2">`
- **cylinder**: `<rect>` + 2 `<ellipse>` сверху/снизу (перспектива)

Каждая нода — `enterG > depthG > [shape + label text]`. Label по центру, multiline через `<tspan>`.

### Цвета

Каждой ноде — цвет из `PALETTE` по индексу. Bloom на нодах с `shape === 'stadium'` (начало/конец).

## Этап 4: Рёбра (edges)

Кривые Безье между нодами. Расчёт точек входа/выхода:

```js
function _edgePoints(a, b) {
  // Выход из a: нижний центр (для direction TB)
  const aOut = { x: a.x, y: a.y + a.h / 2 };
  // Вход в b: верхний центр
  const bIn = { x: b.x, y: b.y - b.h / 2 };
  // Для diamond: 4 порта (top/right/bottom/left)
  return { aOut, bIn };
}
```

Кривая: `M x1,y1 C x1,y1+dy x2,y2-dy x2,y2` (cubic bezier, dy = расстояние * 0.4).

Стрелка: маркер `#arrow-head` (из MindMap). Лейбл на середине кривой.

Для параллельных рёбер между теми же нодами — смещение control points (как в Mermaid Flow).

  const nodeEl = e.target.closest('[data-node-id]');

  cancelAnimationFrame(_inertiaRaf);
  _dragging = false; // блокируем pan
  _dragNode = nodeEl.dataset.nodeId;
  const node = _nodes.find(n => n.id === _dragNode);
  // ... вычислить offset, начать drag
});

window.addEventListener('mousemove', e => {
  if (!_dragNode) return;
  // Обновить позицию ноды в _nodes, перерисовать рёбра
});

window.addEventListener('mouseup', () => {
  _dragNode = null;
});
```

**Важно:** mousedown на ноде → drag ноды, mousedown на пустом месте → pan. Разводка через `e.target.closest('[data-node-id]')`.
