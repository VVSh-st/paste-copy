## Step 4: Implement mini-chat routing for chat modes

When mode is in `_chatModes`:
1. Call `MiniChat.newSession()`, `MiniChat.open()`
2. Add system message (e.g. "Позитивные инструкции...")
3. Stream response into mini-chat via `MiniChat.appendChunk()`
4. On completion: `MiniChat.finalizeLastMessage(result)`
5. Return early (skip diff panel entirely)

This mirrors `_runOnPreview` logic but gets text from the block textarea.
